    Comienza Rápidamente
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Instala y configura tu sistema POS en minutos
  </p>

  <Steps>
    <Step title="Instalar el sistema">
      Descarga POS CDP Lite usando Composer o Git:

      ```bash Composer
      composer create-project mroblesdev/pos-cdp-lite
      ```

      ```bash Git
      git clone https://github.com/mroblesdev/pos-cdp-lite.git
      ```
    </Step>

    <Step title="Configurar la base de datos">
      Crea una base de datos MySQL y configura tu archivo `.env`:

      ```sql
      CREATE DATABASE IF NOT EXISTS `pos-cdp-lite` 
      DEFAULT CHARACTER SET utf8mb4 
      COLLATE utf8mb4_unicode_ci;
      ```

      ```bash .env
      database.default.hostname = localhost
      database.default.database = pos-cdp-lite
      database.default.username = root
      database.default.password = password
      ```
    </Step>

    <Step title="Ejecutar migraciones">
      Ejecuta las migraciones y seeders para crear las tablas e insertar datos iniciales:

      ```bash
      php spark migrate
      php spark db:seed CargaSeeder
      ```
    </Step>

    <Step title="Iniciar el sistema">
      Inicia el servidor de desarrollo:

      ```bash
      php spark serve
      ```

      Accede al sistema en `http://localhost:8080` con las credenciales por defecto:
      - **Usuario:** admin
      - **Contraseña:** admin
    </Step>
  </Steps>
</div>
