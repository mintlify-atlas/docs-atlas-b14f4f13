## ¿Qué es POS CDP Lite?

POS CDP Lite es una aplicación web de código abierto basada en PHP y MySQL diseñada para gestionar las ventas en tu tienda de manera eficiente. Desarrollado con CodeIgniter 4, ofrece una solución completa y profesional para puntos de venta pequeños y medianos.

<Note>
  Este proyecto es completamente gratuito y de código abierto bajo licencia MIT. Puedes usarlo, modificarlo y distribuirlo libremente.
</Note>

## Requisitos del Sistema

<Accordion title="Servidor Web">
  Apache Server 2.4 o superior (también compatible con Nginx)
</Accordion>

<Accordion title="PHP">
  - **Versión:** PHP 8.1 o superior
  - **Extensiones requeridas:**
    - `intl` - Internacionalización
    - `mbstring` - Manejo de cadenas multibyte
    - `json` - Procesamiento de JSON
</Accordion>

<Accordion title="Base de Datos">
  MySQL 5.6 o superior / MariaDB 10.3 o superior
</Accordion>

<Accordion title="Herramientas Opcionales">
  - **Composer** - Gestor de dependencias para PHP
  - **Git** - Control de versiones
</Accordion>

## ¿Para Quién es Este Sistema?

POS CDP Lite es ideal para:

- **Tiendas pequeñas y medianas** que necesitan un sistema POS confiable
- **Emprendedores** que buscan una solución sin costos de licencia
- **Desarrolladores** que quieren personalizar su sistema POS
- **Estudiantes** aprendiendo desarrollo web con CodeIgniter 4
