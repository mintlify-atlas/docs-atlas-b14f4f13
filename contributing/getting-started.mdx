## Proceso de Contribución

<Steps>
  <Step title="Fork del repositorio">
    Haz un fork de [mroblesdev/pos-cdp-lite](https://github.com/mroblesdev/pos-cdp-lite)
  </Step>

  <Step title="Clonar tu fork">
    ```bash
    git clone https://github.com/tu-usuario/pos-cdp-lite.git
    cd pos-cdp-lite
    ```
  </Step>

  <Step title="Crear rama">
    ```bash
    git checkout -b feature/mi-nueva-caracteristica
    ```
  </Step>

  <Step title="Hacer cambios">
    Implementa tu feature o corrección
  </Step>

  <Step title="Commit">
    ```bash
    git add .
    git commit -m "Descripción clara de los cambios"
    ```
  </Step>

  <Step title="Push">
    ```bash
    git push origin feature/mi-nueva-caracteristica
    ```
  </Step>

  <Step title="Pull Request">
    Crea un PR en GitHub con:
    - Descripción clara de los cambios
    - Referencia a issues relacionados
    - Screenshots si aplica
  </Step>
</Steps>

## Reportar Issues

Al reportar un bug, incluye:

- Versión de PHP
- Versión de MySQL
- Pasos para reproducir
- Comportamiento esperado vs actual
- Screenshots o logs de error
