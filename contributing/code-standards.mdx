## CodeIgniter 4 Style Guide

POS CDP Lite sigue las convenciones de CodeIgniter 4.

## Convenciones PHP

### PSR-12

El código sigue PSR-12:

- Indentación con 4 espacios
- Llaves en nueva línea para clases y métodos
- CamelCase para nombres de clases
- camelCase para nombres de métodos

```php
class ProductosModel extends Model
{
    public function buscarProducto($codigo)
    {
        return $this->where('codigo', $codigo)->first();
    }
}
```

### Nombres de Archivos

- Controladores: `PascalCase.php`
- Modelos: `PascalCaseModel.php`
- Vistas: `snake_case.php`

## Seguridad

- Usa validación para toda entrada de usuario
- Escapa salida con `esc()`
- Usa Query Builder, nunca queries raw
- Valida y sanitiza datos
